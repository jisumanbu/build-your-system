# 逐字稿写作

生成口语化、去AI味的视频逐字稿。

## 输入要求

在生成逐字稿之前，需要确认：
1. **Hook**：开头钩子（3秒内抓住注意力）
2. **末尾句**：结尾的一句话（与Hook呼应）
3. **逻辑骨架**：内容结构（每个节点一句话）

## 生成流程

### Step 1：确认骨架

```
Hook: [震惊事实]
  ↓
节点1: [要点] + [数据]
  ↓ 但是
节点2: [转折]
  ↓ 所以
节点3: [结论]
  ↓
末尾句: [一句话收尾]
```

### Step 2：展开内容

每个节点展开为 3-5 句话，保持逻辑连贯。

### Step 3：口语化处理

应用去AI味规则和口语化技巧。

## 去AI味规则

### 黑名单词汇（直接删除或替换）

| 禁用词汇 | 替换为 |
|---------|--------|
| 值得一提的是 | 删除 |
| 令人震惊 | 删除或用具体描述 |
| 不仅...而且 | 拆成两句 |
| 众所周知 | 删除 |
| 毫无疑问 | 删除 |
| 显而易见 | 删除 |
| 首先...其次...最后 | 用更自然的过渡 |

### 口语化替换表

| 书面语 | 口语 |
|--------|------|
| 因此 | 所以 |
| 然而 | 但是 |
| 此外 | 还有 |
| 例如 | 比如 |
| 基于 | 根据 |
| 综上所述 | 所以说 |
| 由于 | 因为 |

### 口语化技巧

1. **加语气词**：嘛、吧、啊、呢
2. **不完整句**：省略主语、用省略号
3. **疑问推进**：用问句代替陈述
4. **口头禅**：说真的、你想想、关键是

## 节奏控制

```
快（Hook）→ 中（展开）→ 快（高潮）→ 中 → 快（结尾）
```

### 节奏标记

在逐字稿中标注节奏：

```markdown
[Hook] (节奏：快)
2021年买房，4年亏了100万。

---

[展开] (节奏：中 → 转折处加速)
我是2021年初买的房...
房价下跌了30%...

但是！（停顿）
我没有选择死扛。

---

[收束] (节奏：加速 → 干脆结束)
所以说...
承认错了，比死扛轻松。
```

## AI 自检清单

生成后必须检查：

- [ ] 删除所有黑名单词汇？
- [ ] 用口语替换书面语？
- [ ] 有停顿和呼吸感？
- [ ] 节奏有快慢变化？
- [ ] 读出来像人说的话？
- [ ] 时长控制在 3-5 分钟（600-1000字）？

## 输出格式

```markdown
# {选题名称}

> 选题：[[选题文件路径]]
> 预估时长：X分X秒
> 字数：XXX

---

[Hook] (节奏：快)
{Hook 内容}

---

[展开] (节奏：中)
{主体内容}

---

[收束] (节奏：快)
{末尾句}
```
